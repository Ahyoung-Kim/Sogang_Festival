### PATH

| path                  | 설명                                |
| --------------------- | ----------------------------------- |
| /                     | 메인 페이지                         |
| /flea-market/:day     | 플리마켓/마당 요일 별 페이지        |
| /flea-market/:flea_id | 플리마켓/마당 각 부스별 세부 페이지 |
| /food-truck/:day      | 푸드트럭 요일 별 페이지             |
| /food-truck/:food_id  | 각 푸드트럭 세부 페이지             |
| /schedule             | 공연 일정 페이지                    |
